# Platforms Index
** This endpoint is currently active **

## Description
Returns all allowed values for apple product lines.

## Endpoint
`/api/v1/params/apple_product_line`

## HTTP Method
GET

## Arguments
None

## Example Response

```
[
  {"id":1,"name":"IPAD"},
  {"id":2,"name":"IPHONE"},
  {"id":3,"name":"ITOUCH"}
]
```
