# Personas Index
** This endpoint is currently active **

## Description
Returns all allowed values for personas.

## Endpoint
`/api/v1/params/personas`

## HTTP Method
GET

## Arguments
None

## Example Response

```
[
  {"id":1,"name":"action game player"},
  {"id":2,"name":"adventure game player"},
  {"id":3,"name":"angry birds player"}
]
```
