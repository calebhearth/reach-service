# Platforms Index
** This endpoint is currently active **

## Description
Returns all allowed values for platform.

## Endpoint
`/api/v1/params/platforms`

## HTTP Method
GET

## Arguments
None

## Example Response

```
[
  {
    'id' : 1
    'name' : 'android',
  },
  {
    'id' : 2
    'name' : 'ios',
  }
  {
    'id' : 3
    'name' : 'windows',
  }
]
```
